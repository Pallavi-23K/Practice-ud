### 2. **GET Request (Retrieve All Items / Specific Item)**

* **URL:**

  * Get all items â†’ `http://127.0.0.1:5000/items`

### 3. **POST Request (Add a New Item)**

* **URL:** `http://127.0.0.1:5000/items`
* **Method:** `POST`
* **Body â†’ raw â†’ JSON (application/json):**

  ```json
  {
    "name": "Item 3",                (# this is the format i can add ...so to create a new id)
    "description": "This is item 3"
  }
  ```
* **Expected Response:**

  ```json
  {
    "id": 3,

    "description": "This is item 3"
  }
  ```

---

### 4. **PUT Request (Update an Existing Item)**

* **URL:** `http://127.0.0.1:5000/items/2`
  (example: update item with id=2)
* **Method:** `PUT`
* **Body â†’ raw â†’ JSON:**

  ```json
  {
    "name": "Updated Item 2",
    "description": "This is the updated description for item 2"
  }
  ```
* **Expected Response:** Updated item JSON.

---

### 5. **DELETE Request (Remove an Item)**

* **URL:** `http://127.0.0.1:5000/items/1`
  (example: delete item with id=1)
* **Method:** `DELETE`
* **Body:** Not required.
* **Expected Response:**

  ```json
  {
    "result": "Item deleted"
  }
  ```
